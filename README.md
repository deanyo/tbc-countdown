# TBC Countdown

[![pages-build-deployment](https://github.com/deanyo/tbc-countdown/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/deanyo/tbc-countdown/actions/workflows/pages/pages-build-deployment)

Countdown timer for World of Warcraft: Burning Crusade Classic Anniversary Edition.

**Live:** [dnyo.co.uk/tbc-countdown](https://dnyo.co.uk/tbc-countdown/)

## Features

- Prepatch countdown (January 13/14, 2026)
- TBC launch countdown (February 5, 2026 at 3:00 PM PST)
- Multiple timezone support: EU (CET/GMT), US (PST/EST)
- Real-time countdown updates

## Key Dates

- **Prepatch**: January 13, 2026 (after regional maintenance)
- **TBC Launch**: February 5, 2026 at 3:00 PM PST globally
- **Arena Season 1**: February 17, 2026
- **Raids Unlock**: February 19, 2026 (Karazhan, Gruul's Lair, Magtheridon's Lair)
